express에서의 템플릿 엔진 사용





node.js로 서버를 구현하고 클라이언트에게 보낼 웹페이지를 구현하는 과정에서, 대부분의 경우에는 HTML과 CSS를 사용한다. 그러나, jade나 pug등을 사용할 경우에는 html을 사용할 때와는 달리 템플릿 엔진을 사용해서 jade나 pug를 html로 렌더링해야 한다.



템플릿 엔진을 node.js에서 사용하는 자세한 방식은 아래의 링크를 통해서 확인할 수 있다.
