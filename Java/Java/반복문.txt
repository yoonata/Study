반복문



이름에서부터 알 수 있듯이, 반복문이란 반복 작업을 쉽게 하도록 도와주는 도구이다.



자바에서 반복문은 for 문, while 문, do-while 문의 세 가지뿐이다. (for-each 문의 경우 for 문의 응용 형태이므로 같은 반복문으로 취급하자)



for 문은 반복하는 횟수가 정해진 경우에 주로 이용하며, while 문과 do-while 문은 반복 횟수가 정해지지 않고 조건에 따라 반복이 계속되거나 중단되는 경우에 이용한다. while 문은 반복 조건을 처음부터 따지는 경우에, do-while 문은 반복 조건을 나중에 따지는 경우에 적합하다.


 


또한, for 문의 경우 내부에 조건식이 없어도 컴파일 오류가 생기지 않는 반면에, while 문과 do-while 문의 경우 조건식이 없으면 컴파일 오류가 생기게 된다.
